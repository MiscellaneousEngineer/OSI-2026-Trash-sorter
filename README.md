# OSI-2026-Trash-sorter
Repo for the French Olympiade de SI project

Team project aimed at making a autonomus recycling sorting machine.

We are using a combination of multiple technologies to sort trash effectively, those used in our prototype are an Eddy Current sensor, laser spectrometry to differentiate between glass and plastic, and a machine learning model in conjunction with these sensors, to provide the main data input.
